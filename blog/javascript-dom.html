Explaination of control flow and loops using an every day example

A cities road network has signs at junctions indicating what is in each direction. We can think of them as 'if' statements "if" you want to go to the North Shore then take this road. "While" you are on this road don't go more than 80km/h.


What is the DOM? ...and an example of how you might interact with it

The DOM can be thought of as a tree structure with nodes. One example of how we interact with it is by going down different branches to find the elements we want.


Difference between accessing data from arrays and objects

An array is a list of items and we can access each item by using square brackets containing the position of the item we want. Objects have properties that contain values, and you can access these properties by typing objectName.propertyName.


What are functions? Why are they useful?

Functions have code inside of them that we can execute by calling the function. Being able to execute the code just by calling the function is useful as it gives us a level of abstraction.
